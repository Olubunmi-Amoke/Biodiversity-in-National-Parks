# Biodiversity-in-National-Parks

The columns in the species dataset include:

category - The category of each species
scientific_name - The scientific name of each plant/animal
common_names - The common name of each plant/animal
conservation_status - Each animal/plant's conservation status


The columns of observations dataset are:

scientific_name - The scientific name of each plant/animal
park_name - The name of the national parks
observations - The number of observations within 7 days
